# CarService

**Факултетен номер:** 2401322012

## Име на проекта

CarService

## Описание на проекта

CarService е уеб приложение за управление на автосервиз. Системата позволява работа с информация за автомобили, клиенти и сервизни услуги. Проектът е разделен на два основни компонента:

* **CarService.Api** – REST API, което предоставя достъп до данните и бизнес логиката.
* **CarService.Mvc** – уеб интерфейс (MVC приложение), чрез който потребителите могат да използват функционалностите на системата през браузър.

Чрез приложението могат да се създават, преглеждат и управляват записи за обслужване на автомобили и свързаните с тях данни.

## Инструкции за инсталация

1. Инсталирайте **Visual Studio 2022** (или по-нова версия) с включен пакет **ASP.NET and web development**.
2. Разархивирайте проекта.
3. Отворете файла **CarService.sln** във Visual Studio.
4. Уверете се, че всички зависимости са възстановени и стринга на сървъра.

## Стартиране на проекта

1. Отворете решението **CarService.sln**.
2. Задайте **Multiple Startup Projects**:

   * `CarService.Api`
   * `CarService.Mvc`
3. Стартирайте проекта.
4. Уеб приложението ще се отвори в браузъра и ще комуникира с API сървъра.
